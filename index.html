Questing.lua

Questing.lua is a script for PROShine that plays Pokemon Revolution Online for you from the very Start to as far as possible.

How To Use

To use Questing.lua, simply download the latest version, unzip it in your PROShine Scripts folder and, optionally, edit the configuration file config.lua. Then open the Questing.lua root file from PROShine.

How Far Does It Go

Each official release comes with a changelog on the release page.

If you cannot wait for the next official release, you can check the Quests files of each region, alternatively you can check the QuestManager.lua where all the quests are loaded and sorted.

Contribute

You can create your own quests. A template quest file is updated frequently to help you.

If you have a good knowledge of Lua you can improve the structure of the code and create pull requests. However I would advise creating a ticket on the issue tracker to discuss new ideas.

Questing.lua could also use some other advanced scripts to manage the fights, the team setup or a global path finding script. If you want to work on such complicated and heavy scripts, do not hesitate to contact me, preferably via the issue tracker.
